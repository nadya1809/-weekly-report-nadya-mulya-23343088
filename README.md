# Jobsheet 06

Jobsheet 06 membahas tentang struktur data stack dalam pemrograman C. Stack adalah kumpulan data yang mengikuti prinsip Last In First Out (LIFO), di mana elemen terakhir yang dimasukkan akan menjadi yang pertama dikeluarkan. Pada implementasinya, stack dapat menggunakan array atau linked list. Pada array, perlu diperhatikan batas maksimum elemen yang dapat disimpan agar tidak terjadi overflow. Operasi dasar pada stack meliputi push (menambahkan elemen), pop (menghapus elemen), peek (melihat elemen teratas tanpa menghapus), isFull (memeriksa apakah stack penuh), isEmpty (memeriksa apakah stack kosong), dan clear (mengosongkan stack). Beberapa contoh program juga diberikan untuk implementasi stack menggunakan array, linked list, dan double stack. Salah satu contoh program yang menarik adalah pembalik kata dengan menggunakan stack, yang memanfaatkan rekursi untuk membalikkan urutan karakter dalam sebuah kata.

# Jobsheet 07

Jobsheet 07 ini membahas tentang queue atau antrian dalam pemrograman, yang menggunakan prinsip First In First Out (FIFO). Konsep antrian ini diilustrasikan seperti membeli tiket bioskop, di mana orang pertama yang masuk antrian adalah yang pertama mendapat layanan. Antrian dapat diimplementasikan menggunakan array atau linked list, dengan operasi dasar seperti enqueue (menambahkan elemen), dequeue (menghapus elemen), isEmpty (memeriksa apakah antrian kosong), isFull (memeriksa apakah antrian penuh), dan peek (melihat elemen paling depan tanpa menghapusnya). Implementasi queue menggunakan array memungkinkan penambahan dan penghapusan elemen dengan memanfaatkan variabel front dan rear, sementara implementasi dengan linked list memungkinkan manajemen memori yang lebih fleksibel. Salah satu contoh program yang diberikan adalah implementasi queue menggunakan array dan linked list dalam bahasa C.

# Jobsheet 08

Jobsheet 08 ini membahas tentang bubble sort dan insertion sort dalam pemrograman C. Bubble sort adalah algoritma pengurutan dengan cara melakukan penukaran data secara terus menerus sampai tidak ada lagi perubahan, sehingga data terurut. Sedangkan insertion sort bekerja dengan cara membandingkan elemen yang akan diurutkan dengan elemen yang sudah diurutkan satu per satu, kemudian memasukkan elemen tersebut ke posisi yang tepat. Dalam praktikum, mahasiswa akan belajar mengimplementasikan kedua algoritma ini menggunakan bahasa pemrograman C dengan IDE DevC++.

# Jobsheet 09

Jobsheet 09 membahas tentang selection sort dan merge sort dalam pemrograman C. Selection sort adalah algoritma pengurutan yang memilih elemen terkecil dalam sebuah list pada setiap iterasi dan menempatkannya di urutan paling depan. Sedangkan merge sort adalah teknik pengurutan berdasarkan pembagian data menjadi dua bagian, mengurutkan masing-masing bagian, lalu menggabungkannya kembali dalam urutan yang benar. Dalam praktikum ini, mahasiswa akan belajar mengimplementasikan kedua algoritma ini menggunakan bahasa pemrograman C dengan IDE DevC++.
